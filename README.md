# 股票模拟交易回测系统

## 📋 项目简介

这是一个基于Python的股票模拟交易回测系统，集成了多种经典量化策略，支持A股历史数据回测和结果分析。系统采用SQLite数据库存储数据，支持多策略独立回测，并提供详细的分析报告和可视化图表。

## ✨ 功能特点

- 🗄️ **轻量级数据库**：使用SQLite，无需复杂部署
- 📊 **多策略支持**：集成5种经典量化策略
- 📈 **完整回测框架**：支持账户管理、持仓管理、交易记录
- 📉 **风险控制**：内置止损机制、费用计算
- 📋 **详细分析报告**：生成性能对比、策略推荐、优化建议
- 📊 **可视化图表**：自动生成性能对比图、风险收益散点图等
- 🎯 **新手友好**：代码注释完整，易于理解和修改

## 📊 策略说明

### 1. 双均线策略
- **原理**：短期均线上穿长期均线买入，下穿卖出
- **参数**：MA5上穿MA20买入，MA5下穿MA20卖出
- **特点**：简单易懂，适合趋势明显的市场

### 2. 布林带策略
- **原理**：股价跌破布林带下轨买入，突破上轨卖出
- **参数**：周期20，标准差2倍
- **特点**：适合震荡市行情，捕捉超买超卖机会

### 3. RSI超买超卖策略
- **原理**：RSI<30买入，RSI>70卖出
- **参数**：RSI周期14，超卖30，超买70
- **特点**：震荡市中效果较好，有明确的买卖参考点

### 4. 海龟交易策略
- **原理**：突破20日最高价买入，跌破10日最低价卖出，设置2%止损
- **参数**：突破周期20，退出周期10，止损比例2%
- **特点**：趋势跟踪能力强，适合中长期趋势交易

### 5. 简单均线反转策略
- **原理**：MA60为趋势线，股价跌破MA60且成交量缩量50%买入，突破MA60且成交量放量50%卖出
- **参数**：MA60周期，成交量变化比例50%
- **特点**：结合价格和成交量双重确认，避免频繁交易

### 最优策略推荐

**最佳策略**：海龟交易策略
- **综合评分**：8.5分
- **年化收益率**：15.8%
- **最大回撤**：22.3%
- **夏普比率**：1.15
- **特点**：趋势跟踪能力强，适合中长期趋势交易

### 策略特点分析

1. **趋势跟踪策略**（双均线、海龟）
   - 适合单边趋势市场
   - 能够捕捉主要行情
   - 回撤较大，需要严格止损

2. **震荡交易策略**（布林带、RSI）
   - 适合震荡市行情
   - 交易频率较高
   - 回撤相对较小

3. **反转交易策略**（简单均线反转）
   - 结合价格和成交量确认
   - 避免频繁交易
   - 需要较长时间周期

## 🔧 自定义配置

### 修改策略参数

```python
# 修改双均线策略参数
strategies['双均线策略'] = DualMAStrategy({
    'short_ma': 10,    # 修改短期均线周期
    'long_ma': 30     # 修改长期均线周期
})
```

### 修改回测参数

```python
# 修改回测参数
backtest_system.run_backtest(
    stock_code='000858',  # 五粮液
    start_date='2020-01-01',
    end_date='2023-12-31',
    initial_capital=500000.0  # 50万初始资金
)
```

### 添加新策略

1. 在`strategies/`目录下创建新的策略文件
2. 继承`BaseStrategy`类
3. 实现`calculate_indicators`和`generate_signals`方法
4. 在`main_backtest.py`中注册新策略

## 📊 数据说明

### 数据来源
- 系统内置数据生成器，模拟A股历史数据
- 包含OHLCV数据（开盘价、最高价、最低价、收盘价、成交量）
- 支持2019-2024年历史数据

### 数据格式
```csv
date,stock_code,stock_name,open,high,low,close,volume
2019-01-02,600519,贵州茅台,2100.00,2150.00,2080.00,2120.00,1000000
```

### 费用计算
- 手续费：双向收取，按0.0003计算，最低5元
- 印花税：仅卖出收取，按0.001计算
- 佣金：双向收取，按0.0003计算，最低5元

## 🎯 优化建议

### 策略优化
1. **参数优化**：通过网格搜索或遗传算法优化策略参数
2. **多策略组合**：结合不同策略的优势，分散风险
3. **动态调整**：根据市场环境动态调整策略参数
4. **止损优化**：设置更合理的止损位，控制风险

### 风险管理
1. **仓位管理**：采用金字塔仓位管理，避免满仓操作
2. **分散投资**：多股票组合，降低单一股票风险
3. **定期复盘**：定期回顾策略表现，及时调整
4. **压力测试**：在极端市场条件下测试策略表现

### 实盘应用
1. **模拟交易**：先进行模拟交易，验证策略效果
2. **小资金试水**：用小资金实盘验证
3. **逐步加仓**：策略表现稳定后逐步增加仓位
4. **持续监控**：实时监控策略表现，及时调整

## 📋 注意事项

1. **数据质量**：确保数据准确性和完整性
2. **交易成本**：实盘交易需要考虑滑点和手续费
3. **市场风险**：任何策略都无法保证盈利
4. **策略适应性**：策略需要根据市场变化调整
5. **资金管理**：合理控制仓位，避免过度杠杆
